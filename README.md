# Wstęp
Jest to repozytorium zawierające zadania wykonywane na rzecz laboratorium z przedmiotu WSI - Wstęp do Sztucznej Inteligencji - na Politechnice Warszawskiej w semestrze zimowym 2025 roku.

# Nawigacja
Wykonanie poszczególnych zadań _(np. optymalizacja, predykcja, sieci neuronowe)_ są umieszczone na odpowiednich gałęziach.
